Ajout d'un selfie obligatoire dans l'upload de la pièce d'identité
Endpoint mis à jour
PUT /api/artisans/{uid}/identite/
Headers: Authorization: Bearer <token>
Content-Type: multipart/form-data
Champs :
Champ	Type	Requis	Description
type_piece	string	oui	cni, passeport ou permis
recto	file	oui	JPG/PNG, max 5 Mo
selfie	file	oui	Selfie avec la pièce d'identité visible à côté, JPG/PNG, max 5 Mo
verso	file	non	JPG/PNG, max 5 Mo
Modifications dans le formulaire
- Ajouter un champ de téléchargement "Selfie avec votre pièce d'identité" avec la consigne : "Prenez une photo de vous avec votre pièce d'identité visible à côté de votre visage"
- Le champ est obligatoire — le formulaire ne peut pas être soumis sans
- Mêmes validations que les autres images (formats JPG/PNG, max 5 Mo)
Réponse (GET / PUT)
{
  "uid": "uuid",
  "type_piece": "cni",
  "recto": "https://.../recto.jpg",
  "selfie": "https://.../selfie.jpg",
  "verso": "https://.../verso.jpg",
  "statut": "EN_ATTENTE",
  ...
}
Le champ selfie est présent dans la réponse de consultation comme dans la réponse après upload.
